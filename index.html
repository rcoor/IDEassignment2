Visualization

We have chosen to make a bar chart and a multi line visualization using D3.
The multi line visualization is created by adapting our data to an existing visualization we found online.
( https://bl.ocks.org/d3noob/4db972df5d7efc7d611255d1cc6f3c4f)

Two different types of visualizations built using D3.

Yellow indicates that the year's temperature is above the average temperature since 1880 till 2015.
Blue indicates that the year's temperature is below the average temperature since 1880 till 2015.

Motivation

Since we both live in Copenhagen, we find it interesting to see how the global warming has had an effect on the temperatures in this city.
We have chosen to visualize this by creating a bar plot and a Multi line plot.

Method

Dataset: We have chosen to visualize the dataset for Copenhagen, on how the weather has changed during the last 135 years.
Bar plot:
In the bar plot we have plotted the average temperatures per year in celsius from 1880 to 2015.
This is done to get an nice and easy overview over the overall temperatures per year. By this we can see if there is a pattern that shows us that the temperatures gets higher or lower.
Multi line plot
The Bar plot doesn't show if the the temperatures per month, and if we are dealing with global warming we could find out that the cold months are getting colder, and the warm months are getting warmer.
This is why we chose the multi line plot, since it gives us more information, and we are still able to get a good overview of the data.
