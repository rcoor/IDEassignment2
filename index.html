Project 2 - Temperature Data

Introduction

In this assignment we want to answer the question: How has the weather changed over the past 100 years?.
To answer this question we will use temperature data for Copenhagen, Denmark provided by the NASA.

Visualization

To investigate how the weather has changed over the past 100 years, we will do 2 different visualization using D3.
The first plot is a bar plot of how the average temperatures each year has changed in celsius.
The Bar chart doesn't show the temperatures per month, and if we are dealing with high changes when it is winter and summer we could find out that the cold months are getting colder, and the warm months are getting warmer.
The second plot is therefor a multi line plot of how the average temperatures each month has changed for each year in celsius.
The multi line plot is created by adapting our data to an existing visualization we found online.
( https://bl.ocks.org/d3noob/4db972df5d7efc7d611255d1cc6f3c4f)

Bar plot

Yellow indicates that the year's temperature is above the average temperature since 1880 till 2015.
Blue indicates that the year's temperature is below the average temperature since 1880 till 2015.

The bar plot gives a good overview on how the overall changes in temperature per year has changed. There are a general pattern of increasing temperatures, however it is hard to tell if the changes are more significant, since maybe the cold months are getting colder and the warm months are getting warmer.
